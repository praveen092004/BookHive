
# 🛒 BookHive - Online Bookstore 
  
A modern, responsive online bookstore showcasing Indian authors and bestsellers. 
  
## 🛒 Features 
  - Browse and search books by category - Detailed book pages with descriptions - Shopping cart with quantity controls - User signup and login pages - Fully responsive design (mobile, tablet, desktop) 
  
## 🛒 Built With 
  - **HTML5** - Semantic markup - **CSS3** - Custom styling, Flexbox, Grid, animations - **JavaScript (ES6+)** - Interactivity, localStorage, DOM manipulation
